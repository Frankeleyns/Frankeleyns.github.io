安装 JAVA环境(jdk1.8)

第一步

下载jdk1.8

进入Oracle官网下载页面

如果你下的很慢的话，我给你贴心的准备好了我下好了的 点我(提取码：guk0)

下好后可以用WinScp或XFTP自行上传到自己的服务器目录，如果不会的话，我建议你百度

我这里将JDK传到了我的~/software目录，我建议你们也可以自己建个software目录专门存放要安装的软件包

这里可以看到我的software目录已经有我上传的jdk了

第二步

解压已经传到服务器的jdk文件

使用tar命令将software目录下的jdk解压到app目录

tar -zvxf jdk-8u241-linux-x64.tar.gz  -C ~/app

解压完进入查看一下

[hadoop@hadoop000 ~]$ ls
app   Desktop    Downloads  maven_resp  Pictures  README.txt  software   t.txt
data  Documents  lib        Music       Public    shell       Templates  Videos
[hadoop@hadoop000 ~]$ cd app
[hadoop@hadoop000 app]$ ls
hadoop-2.6.0-cdh5.15.1  hive-1.1.0-cdh5.15.1  jdk1.8.0_241  tmp
[hadoop@hadoop000 app]$ cd jdk1.8.0_241/
[hadoop@hadoop000 jdk1.8.0_241]$ pwd
/home/hadoop/app/jdk1.8.0_241

可以看到我的jdk已经解压到app目录了，路径是/home/hadoop/app/jdk1.8.0_241

第三步

配置环境变量

编辑环境变量文件

vim ~/.bash_profile

找个地方，添加下面这段

export JAVA_HOME=/home/hadoop/app/jdk1.8.0_241
export PATH=$JAVA_HOME/bin:$PATH

使环境变量生效

 source ~/.bash_profile

使用java命令，验证生效情况

java

出现，如图所示情况，安装成功😀

总结

在配置环境变量的时候，一般网上都是

vim /etc/profile

而我的是.bash_profile，这两者的区别再去

/etc/profile：这个配置文件是对所有系统用户生效

.bash_profile：这个配置文件是对当前登录生效
